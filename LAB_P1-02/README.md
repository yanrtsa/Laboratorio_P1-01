# Laboratório P1

## Como executar o código

1. Instale as dependências necessárias:


pip install numpy pandas


2. Execute o arquivo Python:


python main.py


3. O programa irá:

- Criar um vocabulário simples
- Converter uma frase em tokens
- Gerar embeddings
- Executar o Encoder Transformer com 6 camadas
- Mostrar as saídas das camadas e o resultado final no terminal

## Observação

Ferramentas de Inteligência Artificial foram utilizadas apenas para tirar dúvidas durante o desenvolvimento do laboratório.
